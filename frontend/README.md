# VeritasAI Frontend

This is the user interface for the VeritasAI application, built with React, Vite, and Tailwind CSS.

To get the frontend running:

1.  **Navigate to this folder:**
    ```bash
    cd frontend
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Start Development Server:**
    ```bash
    npm run dev
    ```

The application will start at `http://localhost:5173`.
